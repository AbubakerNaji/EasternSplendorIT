How to Export as PNG:

  1. Right-click on the logo size you want to export
  2. Select "Inspect" or "Inspect Element"
  3. In the developer tools, find the <svg> element
  4. Right-click on the SVG element in the inspector
  5. Select "Capture node screenshot" (Chrome/Edge) or use a screenshot tool
  6. Alternatively, you can use online tools like CloudConvert to convert the SVG file to PNG

Tip: The SVG file (logo.svg) can be directly converted to PNG using design software like Adobe Illustrator, Inkscape, or online converters.
